# <a name="psworkflow-module"></a>PSWorkflow 模組

本節包含 **PSWorkflow** 模組的說明主題，此模組包含支援 Windows PowerShell 工作流程功能的 Cmdlet。 **PSWorkflow** 模組是與 Windows PowerShell 5.1 一起安裝。

> [!NOTE]
> 只有在 Windows PowerShell 5.1 中才支援工作流程。

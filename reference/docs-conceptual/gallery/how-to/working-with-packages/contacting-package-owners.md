# <a name="contacting-package-owners"></a>連絡套件擁有者

若要連絡特定套件的擁有者，請瀏覽到套件詳細資料頁面。
左側功能表列中會有連絡擁有者連結。
按一下該連結，您就會前往新的頁面。
您可以從這裡傳送訊息。

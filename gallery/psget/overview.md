# <a name="powershellget-overview"></a>PowerShellGet 概觀

PowerShellGet 模組包含來自 https://www.PowerShellGallery.com 及其他私用存放庫，用於探索、安裝、更新及發行 PowerShell 成品 (例如模組、DSC 資源、角色功能與指令碼) 的 Cmdlet。

# <a name="contacting-item-owners"></a>連絡項目擁有者

若要連絡特定項目的擁有者，請瀏覽到項目詳細資料頁面。
左側功能表列中會有連絡擁有者連結。
按一下該連結，您就會前往新的頁面。
您可以從這裡傳送訊息。

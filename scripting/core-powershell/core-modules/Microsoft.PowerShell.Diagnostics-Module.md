# Microsoft.PowerShell.Diagnostics 模組
本節包含 **Microsoft.PowerShell.Diagnostics** 模組的說明主題。 **Microsoft.PowerShell.Diagnostics** 模組包含可管理事件記錄檔、效能計數器與 Windows 事件追蹤 (ETW) 的 Cmdlet。 此模組是與 Windows PowerShell 一起安裝。

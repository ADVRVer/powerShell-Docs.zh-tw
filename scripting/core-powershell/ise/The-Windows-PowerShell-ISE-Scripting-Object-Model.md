# Windows PowerShell ISE 指令碼物件模型
  Windows PowerShell® 整合式指令碼環境 (ISE) 透過基礎的指令碼物件模型公開其功能，該模型可讓使用者撰寫 Windows PowerShell 指令碼，以操作 Windows PowerShell ISE 的各種視覺及功能層面。 指令碼物件的屬性及其方法皆專為此用途而定義。

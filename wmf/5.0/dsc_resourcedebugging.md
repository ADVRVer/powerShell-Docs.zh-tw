# <a name="dsc-resource-script-debugging"></a>DSC 資源指令碼偵錯

WMF 5.0 支援在目標節點上執行時，偵錯 DSC 資源指令碼。
如需詳細資訊，請參閱[偵錯 DSC 資源](https://msdn.microsoft.com/powershell/dsc/debugresource)。

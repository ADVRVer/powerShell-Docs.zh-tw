# <a name="support-for-mixed-refreshmode"></a>支援混合的 RefreshMode

當您使用部分設定時，您就可以用 `RefreshMode` 定義每個個別的部分設定。
如需詳細資訊，請參閱[部分設定](https://msdn.microsoft.com/powershell/dsc/partialconfigs)。

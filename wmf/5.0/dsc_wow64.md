# WOW64 支援設定關鍵字

64 位元電腦上的 WOW64 現在支援設定關鍵字。 這代表可以定義 DSC 設定，以及可於 32 位元處理序中編譯，例如在 64 位元電腦上執行的 Windows PowerShell ISE (x86)。

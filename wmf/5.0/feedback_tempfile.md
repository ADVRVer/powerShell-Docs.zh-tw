# New-TemporaryFile
有時候在您的指令碼中，您必須建立暫存檔案。 您可以使用 **New-TemporaryFile**Cmdlet 輕鬆執行這項操作︰

PS C:\\&gt; $tempFile = New-TemporaryFile

PS C:\\&gt; $tempFile.FullName

C:\\Users\\slee\\AppData\\Local\\Temp\\tmp375.tmp

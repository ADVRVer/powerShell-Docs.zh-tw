# <a name="cmdlet-development-guidelines"></a>Cmdlet 開發指導方針

在本節中的主題提供可用來產生格式正確的指令程式的開發指導方針。 藉由提供 Windows PowerShell 執行階段和按照下列指導方針的一般功能，您可以開發強大的指令程式，其最少的工作，並提供使用者一致的體驗。 此外，您將會減少測試負擔，因為一般功能不需要重新測試。

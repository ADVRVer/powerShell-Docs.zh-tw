# <a name="typename-element-for-selectioncondition-for-entryselectedby-for-enumerableexpansion-format"></a>EnumerableExpansion 之 EntrySelectedBy 的 SelectionCondition 的 TypeName 元素 (格式)

指定觸發條件的.NET 型別。

組態項目 DefaultSettings 項目 （格式） EnumerableExpansions 項目 （格式） EnumerableExpansions 項目 （格式） EntrySelectedBy 項目為 EntrySelectedBy EnumerableExpansion （格式） SelectionCondition 項目針對 EnumerableExpansion （格式） 的 EntrySelectedBy SelectionCondition EnumerableExpansion （格式） 類型名稱項目

```xml
<TypeName>Nameof.NetType</TypeName>
```

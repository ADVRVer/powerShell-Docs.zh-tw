# <a name="wraptables-element-format"></a>WrapTables 元素 (格式)

指定表格儲存格的資料會移至下一行，是否資料的長度超過資料行的寬度。

組態項目 （格式） DefaultSettings 項目 （格式） WrapTables 項目 （格式）

```xml
<WrapTables/>
```

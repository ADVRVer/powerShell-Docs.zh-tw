# <a name="getproc03-code-samples"></a>GetProc03 程式碼範例

以下是 GetProc03 cmdlet 範例的程式碼範例。 這是`Get-Process`cmdlet 的範例中所述[加入參數，該程序管線輸入](../cmdlet/adding-parameters-that-process-pipeline-input.md)。 這`Get-Process`cmdlet 會使用`Name`接受來自管線的物件中，輸入的參數會從本機電腦，根據提供的名稱，擷取程序資訊，並接著會處理序的相關資訊顯示在命令列。

> [!NOTE]
> 您可以下載C#使用 Microsoft Windows 軟體開發套件的 Windows Vista 和.NET Framework 3.0 執行階段元件這個 Get-proc cmdlet 的原始程式檔 (getprov03.cs)。 如需下載指示，請參閱[如何安裝 Windows PowerShell 並下載 Windows PowerShell SDK](/powershell/developer/installing-the-windows-powershell-sdk)。
>
> 已下載的原始程式檔位於 **\<PowerShell 範例 >** 目錄。
